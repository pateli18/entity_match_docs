### 1. Install the Application in Your Workspace
<Info>This only needs to be done once per workspace</Info>

Search for "EntityMatcher" in the [Google Workspace Marketplace](https://workspace.google.com/marketplace/search/EntityMatcher) and `Install` the application.

<Frame caption="Install EntityMatcher in Google Sheets">
  <img src="/screenshots/gsheets/plugin_install.png" alt="Install EntityMatcher in Google Sheets" />
</Frame>

### 2. Enable the Plugin in Your Worksheet
<Info>This only needs to be done once per worksheet</Info>

Click `Extensions > Add-ons > Manage add-ons` in the menu bar.

<Frame caption="Manage Add-Ons">
  <img src="/screenshots/gsheets/extensions_manage_add_ons.png" alt="Manage Add-Ons" />
</Frame>

Then click the three dots in the top right corner of the EntityMatcher add-on and select `Use in this document`.

<Frame caption="Use in this document">
  <img src="/screenshots/gsheets/plugin_enabled.png" alt="Use in this document" />
</Frame>

<Warning>You may need to exit and re-open the worksheet after enabling it to have the changes take effect</Warning>

### 3. Get and Set Your API Key
<Info>This only needs to be done once per workspace</Info>

<Snippet file="api-key.mdx" />

Once you've retrieved the API key, add it to the plugin by clicking `Extensions > EntityMatcher > API Key` in the menu bar.

<Frame caption="Open API Key Dialog">
  <img src="/screenshots/gsheets/menu_api_key.png" alt="Open API Key Dialog" />
</Frame>

Paste in your API Key and click `Ok`.

<Frame caption="Add API Key">
  <img src="/screenshots/gsheets/api_key_dialog.png" alt="Add API Key" />
</Frame>
